Node/Express setup
===

Step 1: setup
---

- create folder
- open in VS Code
- git init
- 'npm init --yes'
- 'npm install express'
- create .gitignore with:
    - '/node_modules
- create 'server' folder
- create `server.js` in `server` folder
- create `server/public` folder
- in `public`: create `index.html`, `scripts` folder, `vendors` folder
